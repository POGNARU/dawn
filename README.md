# 아침 글감 · dawn

> 고요히, 마음에 머무는 새벽의 읽을거리

**→ [pognaru.github.io/dawn](https://pognaru.github.io/dawn/)**

포근나루(POGNARU STUDIO)의 햇살이 매일 아침 만나는 **글감 다이제스트 아카이브**입니다.
글쓰기와 창작, 융 심리학과 삶의 의미, 고전과 인문, 그리고 밀도 있는 문장의 한국 문학 — 뉴스가 아니라 **책의 결을 닮은 깊은 에세이**를 골라 한 자리에 모읍니다.

## 결 — 여명(dawn)

밤이 옅어지고 아침이 트는 시간의 빛을 담았습니다.

| 색 | |
|---|---|
| 잉크 네이비 | `#1B2440` |
| 미드나잇 | `#4A4373` |
| 박명 라일락 | `#B9A7C9` |
| 여명 살구 | `#F6C9A0` |
| 금빛 | `#E3A954` |
| 안개 화이트 | `#F3F1EC` |

글꼴은 Song Myung(헤더)과 Nanum Myeongjo(본문).

## 구조

- `index.html` — 다이제스트 모음 (`날짜 · 한 줄 설명 · 읽기 →`)
- `entries/YYMMDD.html` — 그날의 글감 (주제별 제목 · 요약 · 원문 링크)
- `assets/style.css` — 여명 디자인 전담

매일 새벽, 새 글감이 한 줄씩 이 모음 맨 위에 더해집니다.
