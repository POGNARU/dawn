20화: 배포 자동화 — 깃 푸시 → GitHub Pages 자동 배포

코드를 고치고 깃에 푸시만 하면, 정해둔 자동화(GitHub Actions)가 알아서 사이트를 다시 지어 올린다. 사람이 서버에 직접 손댈 일이 없다.

실습: 오늘 이 새벽갈피 페이지가 푸시된 뒤 https://pognaru.github.io/dawn/ 에서 실제로 바뀐 걸 확인해보자.
