6화: Git 연동 — 상태 확인·커밋

Claude Code는 git 명령도 직접 실행할 수 있다. 무엇이 바뀌었는지 물으면 git status와 git diff를 돌려 보여주고, "이 변경사항 커밋해줘"라고 하면 알맞은 커밋 메시지까지 만들어 커밋한다. 다만 커밋·푸시처럼 되돌리기 까다로운 작업은 항상 확인을 거치고 진행하는 편이 안전하다.

실습: 무언가 수정한 파일이 있다면 "git status 보여줘"라고 요청해보자. 이어서 "이 변경사항 커밋해줘"라고 하면 Claude가 커밋 메시지를 제안하고 커밋까지 이어간다.
