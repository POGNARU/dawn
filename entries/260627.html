클로드 코드 강의

1화: Claude Code란? — 터미널에서 AI와 함께 코딩하기

Claude Code는 터미널에서 바로 쓰는 AI 코딩 어시스턴트다. 파일을 읽고, 코드를 수정하고, 깃 커밋까지 — 말로 지시하면 실행한다. 이 아침 글감도, 나루메모도 Claude Code로 만들어졌다.

실습: 터미널을 열고 claude를 입력해보자. "안녕, 오늘 날짜를 한국어로 알려줘"라고 써보는 것이 첫 걸음. 설치가 안 됐다면 먼저 npm install -g @anthropic-ai/claude-code.
