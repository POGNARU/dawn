클로드 코드 강의

3화: 파일 읽기 — Read 도구

Claude Code는 파일을 '읽어달라'고 부탁하는 것만으로 내용 전체를 파악한다. "이 파일 읽어줘", "이 폴더에서 .md 파일들 읽어줘" — 구체적인 경로를 함께 주면 더 정확하다. 직접 열지 않아도 Claude가 파일의 맥락을 이해하고 다음 대화로 이어간다.

실습: Claude Code 대화창에 "C:\Users\[내 이름]\Desktop\[파일명] 읽어줘"라고 입력해보자. Claude가 파일 내용을 보여준 뒤, "이 파일에서 [특정 내용]을 찾아줘"로 이어가면 자연스러운 첫 탐색이 완성된다.
