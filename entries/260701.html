클로드 코드 강의

5화: 터미널 명령 실행 — Bash 도구

Claude Code는 파일을 읽고 쓰는 것뿐 아니라, 터미널 명령도 직접 실행할 수 있다. Bash 도구가 그 통로다. "이 폴더 안에 뭐가 있어?"라고 물으면 ls 명령을, "이 패키지 설치해줘"라고 하면 npm install을 대신 돌린다. 사람이 일일이 명령어를 외울 필요 없이, 하고 싶은 일을 말로 전하면 된다.

실습: Claude Code 대화창에 "지금 폴더 안에 파일 목록 보여줘"라고 요청해보자. Claude가 ls(또는 dir) 명령을 실행해 결과를 보여줄 것이다. 이어서 "이 중에 가장 최근에 수정된 파일이 뭐야?"라고 물어도 좋다.
