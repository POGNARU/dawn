클로드 코드 강의

4화: 파일 편집 — Edit/Write 도구

Claude Code는 파일을 "이렇게 바꿔줘"라고 말만 해도 수정한다. Edit 도구는 특정 문자열을 찾아 바꾸고, Write 도구는 파일 전체를 새로 쓴다. 작은 수정에는 Edit, 처음부터 다시 쓸 때는 Write — 이 둘의 차이를 알면 실수를 줄일 수 있다.

실습: Claude Code 대화창에 "이 파일의 3번째 줄을 '오늘도 좋은 하루'로 바꿔줘"라고 요청해보자. 파일 경로와 함께 주면 Claude가 정확히 해당 줄만 수정한다. 수정 후 "원래대로 되돌려줘"도 가능하다.
